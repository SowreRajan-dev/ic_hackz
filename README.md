# ic_hackz

Mentree server scraps the data's from the coding platforms like Skill rack and hacker rack and renders it into the Frontend.



Link for mentor's app : https://www.figma.com/file/ykisk6JkVk4v5sYoV8ymQn/Untitled?node-id=0%3A1

Front end made with React js 
Server is listened with Flask 
Web Scrapping done with selenium
Api requests tested with Postman
And Backend Service is served with firebase
